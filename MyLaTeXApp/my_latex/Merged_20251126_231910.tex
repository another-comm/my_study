\section*{实分析}

\noindent 设 $\alpha(z) = \sum a_n z^n$, $b(z) = \sum b_n z^n$ 是两个收敛半径分别为 $\alpha > \beta > 0$ 的幂级数。
若 $b(z)$ 满足比例法则，即当 $n \to \infty$ 时，$\frac{b_{n-1}}{b_n} \to \beta$。
则乘积 $f(z) = \alpha(z) \cdot b(z)$ 的系数满足
当 $n \to \infty$ 时，有 $f_n \sim \alpha(\beta) b_n$，其中 $\alpha(\beta) \ne 0$。

\noindent \textbf{证明:}
设 $f(z) = \sum f_n z^n$。根据幂级数乘积定义，其系数 $f_n$ 为
\begin{align*}
f_n &= a_0 b_n + a_1 b_{n-1} + \dots + a_n b_0 \\
    &= b_n \left[ a_0 + a_1 \frac{b_{n-1}}{b_n} + a_2 \frac{b_{n-2}}{b_n} + \dots + a_n \frac{b_0}{b_n} \right]
\end{align*}
令 $S_n = f_n$。我们可以将其写成和的形式：
\[ S_n = \sum_{k=0}^n a_k b_{n-k} \]
对于任意固定的正整数 $K < n$，我们可以将 $S_n$ 分为两部分：
\[ S_n = \sum_{k=0}^K a_k b_{n-k} + \sum_{k=K+1}^n a_k b_{n-k} = T_n^{(K)} + R_n^{(K)} \]
由于 $\lim_{n \to \infty} \frac{b_{n-1}}{b_n} = \beta$，可知对任意固定的 $k$，
\[ \lim_{n \to \infty} \frac{b_{n-k}}{b_n} = \lim_{n \to \infty} \left( \frac{b_{n-k}}{b_{n-k+1}} \cdot \frac{b_{n-k+1}}{b_{n-k+2}} \dots \frac{b_{n-1}}{b_n} \right) = \beta^k \]
因此，
\[ \lim_{n \to \infty} \frac{T_n^{(K)}}{b_n} = \lim_{n \to \infty} \sum_{k=0}^K a_k \frac{b_{n-k}}{b_n} = \sum_{k=0}^K a_k \beta^k \]
设 $C = \sum_{k=0}^\infty a_k \beta^k$. 由于 $\beta$ 在 $\alpha(z)$ 的收敛半径内 $(\beta < \alpha)$，此幂级数绝对收敛，故 $C < \infty$。
对 $\forall \varepsilon > 0$，$\exists K_0$ 当 $K > K_0$ 时，我们有 $\left| \sum_{k=K+1}^\infty a_k \beta^k \right| < \varepsilon/2$。

又由于 $\lim_{m \to \infty} \frac{b_m}{b_{m+1}} = \beta$。对给定 $\varepsilon$，$\exists M_0$ 当 $m \ge M_0$ 时，$\left| \frac{b_m}{b_{m+1}} - \beta \right| < \delta$。
待定 $\delta$，使 $\beta+\delta < \left(1+\frac{\varepsilon}{2C}\right)\beta$ (这里 $C$ 指 $\sum_{k=0}^\infty |a_k| \beta^k$)。
故对 $k \ge K+1$ 且 $n-k \ge M_0$ (即 $n > M_0+K+1$)，有
\[ \frac{b_{n-k}}{b_n} = \prod_{j=0}^{k-1} \frac{b_{n-k+j}}{b_{n-k+j+1}} < (\beta+\delta)^k < \left(1+\frac{\varepsilon}{2C}\right) \beta^k \]
当 $K \ge K_0$ 且 $n > N_0$ (对于某个足够大的 $N_0$) 时，对于余项 $R_n^{(K)}$，
\[ \left| \frac{R_n^{(K)}}{b_n} \right| = \left| \sum_{k=K+1}^n a_k \frac{b_{n-k}}{b_n} \right| < \sum_{k=K+1}^n |a_k| \frac{b_{n-k}}{b_n} \]
利用上面的不等式和 $K_0$ 的选取，可得
\[ \sum_{k=K+1}^n |a_k| \frac{b_{n-k}}{b_n} < \sum_{k=K+1}^\infty |a_k| \left(1+\frac{\varepsilon}{2C}\right)\beta^k < \left(1+\frac{\varepsilon}{2C}\right) \frac{\varepsilon}{2} < \varepsilon \]
现在我们考虑 $\left| \frac{S_n}{b_n} - C \right|$：
\begin{align*}
\left| \frac{S_n}{b_n} - C \right| &= \left| \frac{T_n^{(K)}}{b_n} + \frac{R_n^{(K)}}{b_n} - C \right| \\
&\le \left| \frac{T_n^{(K)}}{b_n} - \sum_{k=0}^K a_k \beta^k \right| + \left| \frac{R_n^{(K)}}{b_n} \right| + \left| \sum_{k=K+1}^\infty a_k \beta^k \right| \\
&= \left| \sum_{k=0}^K a_k \left( \frac{b_{n-k}}{b_n} - \beta^k \right) \right| + \left| \frac{R_n^{(K)}}{b_n} \right| + \left| \sum_{k=K+1}^\infty a_k \beta^k \right|
\end{align*}
我们可以选择 $K$ 足够大，使得 $\left| \sum_{k=K+1}^\infty a_k \beta^k \right| < \varepsilon/2$.
对于这个固定的 $K$，可以选择 $N$ 足够大，使得当 $n > N$ 时：
\begin{itemize}
    \item 由于 $\lim_{n \to \infty} \frac{b_{n-k}}{b_n} = \beta^k$ 且 $K$ 是有限的，故 $\left| \sum_{k=0}^K a_k \left( \frac{b_{n-k}}{b_n} - \beta^k \right) \right| < \varepsilon/2$。
    \item 根据前面关于余项的推导，$\left| \frac{R_n^{(K)}}{b_n} \right| < \varepsilon$。
\end{itemize}
因此，对于足够大的 $n$，
\[ \left| \frac{S_n}{b_n} - C \right| < \frac{\varepsilon}{2} + \varepsilon + \frac{\varepsilon}{2} = 2\varepsilon \]
(根据原图片，其界限写为 $\varepsilon + \frac{\varepsilon}{2} + \varepsilon \le \frac{5}{2}\varepsilon$)
\[ \left| \frac{S_n}{b_n} - C \right| < \varepsilon + \frac{\varepsilon}{2} + \varepsilon \le \frac{5}{2}\varepsilon \]
由于 $\varepsilon$ 是任意小的正数，这表明
\[ \lim_{n \to \infty} \frac{S_n}{b_n} = C = \alpha(\beta) \]
证毕。
