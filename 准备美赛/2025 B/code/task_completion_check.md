- [ ] **优化过程集成**：在 `nsga2_solver.py` 中**未集成**随机约束
  - 当前 `W_cap` 是确定性的：`16.0 + 0.3 * ln(1 + invest)` ❌
  - 未考虑季节性波动 ❌

### 问题：
当前优化使用的是**确定性模型**，即：
```python
w_cap_total = 16.0 + w_cap_gain  # 固定值
```

但实际中，供水能力受降雨影响，应该考虑随机性：
```python
W_cap ~ N(16.0 + w_cap_gain, 2.5^2)  # 随机变量
```

### 需要改进：
将优化问题改为**机会约束优化 (Chance-Constrained Optimization)**：
- 约束条件从确定性改为概率性：$P(W_{demand} < W_{cap}) \geq 0.95$
- 在每次评估时，使用蒙特卡洛采样评估约束满足概率
