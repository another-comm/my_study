# Sigmoid归一化（逻辑斯蒂映射）说明

## 📊 数学公式

$$z_{ij} = \frac{1}{1 + e^{-\alpha (\frac{x_{ij} - \bar{x}}{\sigma})}}$$

其中：
- $x_{ij}$：原始指标值
- $\bar{x}$：该指标的均值
- $\sigma$：该指标的标准差
- $\alpha$：Sigmoid曲线陡峭程度参数（默认2.0）
- $z_{ij}$：归一化后的值（0-1范围）

## 🔢 计算步骤

1. **标准化**：$(x_{ij} - \bar{x}) / \sigma$
   - 将数据转换为均值为0、标准差为1的分布

2. **Sigmoid映射**：$1 / (1 + e^{-\alpha \times standardized})$
   - 将标准化后的值映射到0-1范围
   - 通过Sigmoid函数的S型曲线特性，实现非线性映射

## 🎯 核心优势

### 1. **压制极端值**
- **原理**：Sigmoid函数在两端（接近±∞）增长缓慢
- **效果**：极端大值不会得到接近1的分数，极端小值不会得到接近0的分数
- **示例**：
  - Min-Max：最大值 → 1.0，最小值 → 0.0
  - Sigmoid：最大值 → 约0.95，最小值 → 约0.05（取决于α值）

### 2. **放大中端区域细微差距** ⭐ **核心优势**
- **原理**：Sigmoid函数在中间区域（接近0）增长最快
- **效果**：中端国家的细微差距会被放大，更容易区分
- **示例**：
  - 假设UK和France在某个指标上原始值相差10%
  - Min-Max：标准化后可能只差5%
  - Sigmoid：标准化后可能差15%（放大了差距）

### 3. **处理负值指标**
- **优势**：基于均值和标准差的标准化，可以自然处理负值
- **对比**：对数变换无法直接处理负值（如Migration_Idx）

## 📈 参数α的影响

### α = 1.0（平缓）
- Sigmoid曲线较平缓
- 极端值压制效果较弱
- 中端区域放大效果较弱

### α = 2.0（默认）⭐ **推荐**
- 平衡的曲线陡峭程度
- 适度的极端值压制
- 适度的中端区域放大

### α = 3.0（陡峭）
- Sigmoid曲线较陡峭
- 极端值压制效果较强
- 中端区域放大效果较强
- 可能导致中端国家得分过于接近

## 💡 适用场景

### ✅ 适合使用Sigmoid归一化的情况

1. **数据存在极端值**
   - 少数国家在某些指标上远超其他国家
   - 希望压制极端值的影响

2. **需要区分中端国家**
   - 中端国家（如UK、France、Germany）的细微差距很重要
   - 希望放大这些细微差距

3. **有负值指标**
   - 如Migration_Idx（人才流动指数）
   - Sigmoid可以自然处理负值

### ⚠️ 注意事项

1. **参数α需要调优**
   - 默认α=2.0，但可能需要根据数据调整
   - 建议范围：1.0-3.0

2. **结果解释**
   - Sigmoid归一化后的值不是线性关系
   - 需要理解Sigmoid函数的非线性特性

## 📊 与Min-Max和Log-MinMax的对比

| 特性 | Min-Max | Log-MinMax | Sigmoid |
|------|---------|------------|---------|
| **极端值处理** | 放大 | 压缩 | 压制 ⭐ |
| **中端区域** | 线性 | 对数压缩 | 放大 ⭐ |
| **负值处理** | 可以 | 需要特殊处理 | 可以 ⭐ |
| **计算复杂度** | 低 | 中 | 中 |
| **参数调优** | 无 | 无 | α参数 |

## 🔧 代码实现

```python
# Sigmoid归一化实现
alpha = 2.0  # 可调整参数

for col in columns:
    col_data = data[col].values
    mean_val = col_data.mean()
    std_val = col_data.std()
    
    if std_val == 0:
        normalized[col] = 0.5  # 处理标准差为0的情况
    else:
        # 标准化
        standardized = (col_data - mean_val) / std_val
        # Sigmoid映射
        normalized[col] = 1 / (1 + np.exp(-alpha * standardized))
```
